# MVP Spec — Revenue Leak Finder (48-hour demo scope)

## What we are building now
A demoable portal:
- Supabase magic-link login
- Client upload page: upload EOB PDFs / 835 ERAs to private storage
- Readiness meter UI (stubbed)
- Results table UI (sample data)
- Basic admin page (stubbed)

## What we are NOT building in the first 48 hours
- 835 parsing, OCR, scoring engine, PDF report generation
- Stripe payments
- RLS enforcement (we design for it; add later)
